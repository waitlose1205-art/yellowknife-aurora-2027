# 2027 團體商品快速篩選表

## 使用方式

旅行社正式公布 2027 黃刀鎮商品後，逐項填寫本表。任何缺漏不得補猜，先標示 `PENDING_2027_RECHECK`。

| 項目 | 必填內容 | 通過標準 |
| --- | --- | --- |
| 商品名稱 | 旅行社、商品名稱、團號 | 可回溯至官方頁面 |
| 日期與天數 | 台灣出發日、返台日、總天數 | 日期完整 |
| 團費 | 標示團費與幣別 | 可計算每人總額 |
| 必要附加費 | 小費、ESTA、eTA、行李、保險、其他 | 必須加回總成本 |
| 航班 | 航空公司、班號、路線 | 不可只寫參考航班 |
| 轉機 | 機場、時間、次數 | 每方向 2 次以下優先 |
| YZF 抵離時間 | 抵達與離開黃刀鎮時間 | 晚抵不算完整極光夜 |
| 極光安排 | 夜數、是否含抵達夜、補看規則 | A/B 口徑清楚 |
| 飯店 | 實際飯店或同級條件 | 不明確則待重查 |
| 自由時間 | 黃刀鎮市區與 YVR 是否有自主時間 | 能判斷體驗價值 |
| 延誤 / 取消 | 航班取消、天候、改目的地條款 | 不得摧毀核心極光任務 |
| 業者資訊 | 聲譽、重大負面資訊、來源 | 有來源日期 |

## 初步判定

| 條件 | 判定 |
| --- | --- |
| 必要費用後低於 NT$145,000，且 YZF 至少 3 晚 | 強候選 |
| 必要費用後 NT$145,000-150,000 | 條件候選 |
| 必要費用後超過 NT$150,000 | 原則排除 |
| 抵達夜算第一晚但未明確標示 | 退回補查 |
| 中段取消即改目的地且不補看 | 核心任務保護不足 |
| 班號、飯店、YZF 抵離時間未公開 | PENDING_2027_RECHECK |
